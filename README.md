# Featured Events Page

A responsive single-page website showcasing upcoming events with interactive cards.

---

## Technologies Used
- HTML5, CSS3, JavaScript (Vanilla JS)
- Grid & Flexbox for layout
- Images for hero and event cards

---

## How to Run
- Open `index.html` in a browser
- Optionally, use VS Code Live Server or any local server

---

## Features
- Responsive hero and event cards
- Dynamic event rendering from dummy JSON
- Search bar to filter events by name
- Smooth scroll for navbar links
- Hover animations on cards and buttons
- Footer section
